# QMX_quisk_station_sw
Software for remote operating QMX with Quisk. Based on IW7DMH's Remote Morse Key https://iw7dmh.jimdofree.com/other-projects/remote-morse-key/
serial_keyer.py and port.cfg go into a folder in the remote station, edit port.cfg to match the serial port on that system.
https://github.com/davekuk/QMX_quisk_station_sw/blob/main/Remote_Morse_Key_my_v2/Remote_Morse_Key_my_v2.ino is an arduino with Ethernet shield which will connect to your network and a 3.5mm to the QMX CW key jack.
